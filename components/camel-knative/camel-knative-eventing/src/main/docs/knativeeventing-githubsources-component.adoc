== Supported producer operation

- listGithubSources
- getGithubSource
- createGithubSource
- deleteGithubSource

== Knative Eventing GitHubSources Producer Examples

- listGitHubSources: this operation list the Knative Eventing githubsources on a kubernetes cluster

[source,java]
--------------------------------------------------------------------------------
from("direct:list").
	process(new Processor() {
            @Override
            public void process(Exchange exchange) throws Exception {
                exchange.getIn().setHeader(KubernetesConstants.KUBERNETES_NAMESPACE_NAME, "default");
            }
        }).
    toF("knativeeventing-githubsources:///?kubernetesClient=#kubernetesClient&operation=listGithubSources").
    to("mock:result");
--------------------------------------------------------------------------------

This operation list Knative Eventing githubsources in default namespace in your cluster

- createGithubSource:  this operation create a Knative Eventing githubsource in a kubernetes cluster

[source,java]
--------------------------------------------------------------------------------
from("direct:create").
	process(new Processor() {
            @Override
            public void process(Exchange exchange) throws Exception {
                exchange.getIn().setHeader(KubernetesConstants.KUBERNETES_GITHUBSOURCE_NAME, "test");
                exchange.getIn().setHeader(KubernetesConstants.KUBERNETES_NAMESPACE_NAME, "katamari");
                exchange.getIn().setHeader(KubernetesConstants.KUBERNETES_GITHUBSOURCE_EVENT_TYPES, "\"issues\", \"repository\"");
                exchange.getIn().setHeader(KubernetesConstants.KUBERNETES_GITHUBSOURCE_OWNER_REPO, "orgname/reponame");
                exchange.getIn().setHeader(KubernetesConstants.KUBERNETES_GITHUBSOURCE_SECRET, "githubsecret");
                exchange.getIn().setHeader(KubernetesConstants.KUBERNETES_GITHUBSOURCE_ACCESS_TOKEN_KEY, "accessToken");
                exchange.getIn().setHeader(KubernetesConstants.KUBERNETES_GITHUBSOURCE_SECRET_TOKEN_KEY, "secretToken");
                exchange.getIn().setHeader(KubernetesConstants.KUBERNETES_GITHUBSOURCE_API_URL, "https://api.github.com/");
                exchange.getIn().setHeader(KubernetesConstants.KUBERNETES_GITHUBSOURCE_SINK_API_VERSION, "messaging.knative.dev/v1beta1");
                exchange.getIn().setHeader(KubernetesConstants.KUBERNETES_GITHUBSOURCE_SINK_KIND, "Channel");
                exchange.getIn().setHeader(KubernetesConstants.KUBERNETES_GITHUBSOURCE_SINK_NAME, "github");
            }
        }).
    to("kantiveeventing-githubsources:///?kubernetesClient=#kubernetesClient&operation=createGithubSource").
    to("mock:result");
--------------------------------------------------------------------------------

This operation create a Knative Eventing githubsource in your cluster.

== Kubernetes Deployments Consumer Example

[source,java]
--------------------------------------------------------------------------------
from("knativeeventing-githubsources:https://xx.xx.xx.xx:xxxx?trustCerts=true&namespace=default")
	.process(new Processor() {

            @Override
            public void process(Exchange exchange) throws Exception {
                log.info(exchange.getMessage().getBody());
            }
	    });
--------------------------------------------------------------------------------

This consumer will return a list of events on the namespace default for the Kantive Eventing githubsources.
